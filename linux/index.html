Stop the graphical interface: Debian: sudo service gdm or sudo service lightdm stop RPM-based: sudo telinit 3

Remove all the swap files in the current and subsequent dirs
find -iname "*.swp" -exec rm {} ';'
Prompt first
find -iname "*.swp" -ok rm {} ';
Find based on time
find -ctime 3 # inode meta-data updates (file ownership, permissions)
find -mtime # modified
find -atime # Last accessed/read
find -atime 3 # 3 days ago
find -atime +3 # More than 3 days ago
find -atime -3 # Less than 3 days ago
find -{amin, cmin, mmin} 3 # 3 min ago
Find based on size
find -size {+,-}10{c, k, M, G} # bytes (c), kilobytes (k), megabytes (M), gigabytes (G), etc
Find based on type
find -type f # Find files find -type d # Find directory
